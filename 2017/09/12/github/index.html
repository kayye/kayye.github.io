github的操作问题

  • github分支问题
  • Another git process seems to be running in this repository…”问题解决

    造成原因，多数为git push 过程中突然中断而造成的。

根据vs2017的git管理工具的提示信息，打开文件夹选项，打开显示隐藏文件，进入工作区目录下的隐藏文件.git，其中的index.lock文件删除掉，然后重新打开git bash进程，问题解决。
